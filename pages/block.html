What Are Block Elements?

Block elements are the parts of an HTML page or document. HTML stands for HyperText Markup Language. We use block elements to structure a page and specify the different types of text or areas of a page.

HTML ignores spaces and blank lines which are called whitespace. If you leave extra lines nothing happens. Instead, you will show the computer that you want to create a new paragraph using the p tag or you could create a header using the h1 tag. See the photo below as an example.

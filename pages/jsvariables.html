Javascript Variables

JavaScript variables are containers for storing data values. We indicate a variable in javascript with the letters "var". Variables do not have an automatic value so you must define each variable with letters or numbers.

    For example:
  • var name = yourname
  • var age = 45

The letters var indicate that it is a variable and the following word is it's title or name. Then you use the equals sign to define it with a word or number value.

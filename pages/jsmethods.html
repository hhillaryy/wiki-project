JS Methods:

JavaScript methods are the actions performed on objects especially within a function. Methods are functions stored as object properties.

You create an object method by a camel case word like "methodName". Then to access an object method with a syntax that looks like this "objectName.methodName()". Note: methods should not be written in quotation marks. We use them here to specify that this is an example.. Here is a more detailed example:

Here we are going to change our variable to all upper case letters.

We start with the variable "message": var message = "Hello everyone!"

So, to use a method we will write: message.toUpperCase();

This method will display your variable "message" in all capital letters. Like this: HELLO EVERYONE!

Inline Elements

Unlike block elements inline elements do not define the areas or a page or text, nor do they start a new line. Inline elements must be placed within a block element. In other words, they use the block element as their container.

Inline element inlclude: span, a, and img. Divs and spans help define a section of the element within the page. The a (or "anchor") most often provides an anchoring reference point for things like links or images. You may also change a text's display style via inline elements but as you learn more code you will encouraged to make these changes through CSS code instead.
